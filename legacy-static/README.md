# SHAIL Demo Website

A static landing-page prototype for shailai.in / ShellAI.com.

## Files
- `index.html` — page structure and copy
- `styles.css` — visual system, layout, responsiveness
- `script.js` — demo waitlist interaction only

## Launch locally
Open `index.html` in a browser.

## Production notes
- Replace the demo waitlist JavaScript with Supabase, Loops, Resend, or your preferred backend.
- Replace the hero mock cards with real product screenshots or a Ghost Cursor screen recording.
- Keep the top-level page simple; push deeper technical material into product-spec pages.
